CONCEPT

私たちが作りたいのは、
特別な日のための米ではなく、
暮らしの中で、静かに寄り添うごはんです。

強い味ではなく、深い味。
派手さではなく、続くおいしさ。

水や土だけではなく、
この土地で過ごす時間そのものが、
米の一粒に宿ると信じています。

だから今日も、
家族に出せる一杯のために、
この山で米を作っています。

DAILY RECIPE

黒米入りごはん

作り方

  1. 白米を洗い、いつも通り水に浸ける
    （30分〜1時間）
  2. 黒米を軽くすすぐ
    ※洗いすぎなくてOK
  3. 白米に黒米を加える
  4. 水を少し多めに入れ30分～1時間ほど給水させる
    （黒米は水を吸うため）
  5. 普通炊きで炊飯する
  6. 炊き上がったらよく混ぜて完成

おいしく炊くコツ

  • 黒米を増やすほど色が濃くなります
  • 冷めてもおいしい（むしろ旨み出る）
  • おにぎりにすると最高です
BASIC RECIPE

黒米お結び

■ 作り方

  1. 手を水で軽く濡らす
    （ごはんが付きにくくなります）
  2. 手のひらに少量の塩を広げる
  3. ごはんを手にのせ、ふんわりと包むように握る
    ※強く押しすぎないのがコツ。空気を残すと食感がよくなります
  4. お好みで海苔を巻く

■ 材料（2個分）

  • 炊いた黒米ごはん … お茶碗2杯分
  • 塩 … 少々
  • 海苔（お好みで）
  • ※具を入れる場合：梅干し、焼き鮭、味噌など

■ おいしく作るポイント

  • 炊きたてより少し落ち着いたごはんが握りやすいです
  • 黒米入りごはんは冷めても硬くなりにくいです
  • 塩だけでも味が立つのが特徴です
SPECIAL RECIPE

黒米入りお粥

作り方

  1. 米と黒米を軽く洗う
  2. 鍋に米・水を入れ30分ほど浸水
  3. 中火にかけ、沸いたら弱火にして20〜25分煮る
  4. 米がやわらかくなったら完成
  5. お好みで塩をひとつまみ

材料（2人分）

  • 白米 … 1/2合
  • 黒米 … 小さじ1〜2
  • 水 … 約500〜600ml
  • 塩 … 少々（お好み）

おいしく作るポイント

  • 火加減は弱火でゆっくりコトコト煮るのがコツです
  • 水の量でお好みの硬さに調整してください
DAILY RECIPE

黒米炒飯

作り方

  1. フライパンに油を熱し、溶き卵を入れて半熟状にする
  2. ごはんを加えてほぐしながら強めの中火で炒める
  3. 刻んだ具材（ねぎ・ハム等）を入れる
  4. 塩で味を整え、仕上げに醤油を鍋肌から回す
  5. 最後にごま油をひと回しして完成

材料（2人分）

  • 黒米入りごはん … 茶碗2杯分
  • 卵 … 2個
  • 長ねぎ … 1/3本
  • ハム or 焼豚 … 適量
  • 塩 … 少々
  • 醤油 … 小さじ1
  • ごま油 … 小さじ1
  • 油 … 大さじ1

おいしく作るポイント

  • ごはんは温かいものを使うとほぐれやすいです
  • 強めの火で手早く炒めるのがパラパラにするコツです
SEASONAL RECIPE

黒米甘酒ブルーベリースムージー

作り方

  1. 材料をすべてミキサーに入れる
  2. なめらかになるまで攪拌する
  3. 甘さを見て、必要ならバナナを少し追加
  4. グラスに注いで完成

材料（1〜2杯分）

  • 黒米甘酒 … 150ml
  • バナナ … 1本
  • 冷凍ブルーベリー … ひとつかみ
  • 牛乳 or 豆乳 … 50〜100ml
  • 氷 … お好みで

おいしく作るポイント

  • バナナは完熟したものを使うとより甘みが引き立ちます
  • 黒米甘酒の「つぶつぶ感」が残っても食感が楽しめます
FAQ

よくあるご質問

Q. 黒米は毎日食べても大丈夫ですか？

はい、大丈夫です。
白米に少し混ぜるだけなので体への負担はなく、山の暮らしの中でも日常的に食べられてきました。むしろ毎日のごはんに取り入れやすい食材です。

Q. 味にクセはありますか？

強いクセはありません。
ほんのり甘みがあり、もちっとした食感が加わる程度です。白米の味を邪魔しないので、初めての方でも食べやすいです。

Q. どのくらい混ぜればいいですか？

白米2合に対して黒米大さじ1が目安です。
色を濃くしたい場合は少し増やしても大丈夫です。

Q. 子どもでも食べられますか？

問題なく食べられます。
お結びやお粥にすると特に食べやすく、家庭でもよく食べられています。

Q. 保存方法は？

直射日光・高温多湿を避け、密閉して保存してください。
長期保存する場合は冷蔵庫保存がおすすめです。

Q. どんな人が作っていますか？

米良氣農園は、山の中で暮らしながらお米を育てている小さな農園です。
日々の暮らしの延長にある農業を大切にしながら、一つひとつ手をかけて育てています。

Q. 購入前に相談できますか？

もちろん可能です。
食べ方や量の相談など、気軽にお問い合わせください。
